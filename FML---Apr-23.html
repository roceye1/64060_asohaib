centroids_A <- aggregate(partA, list(clusters_A), mean)[, -1]

assign_clusters <- function(data, centroids) {
  dists <- as.matrix(dist(rbind(centroids, data)))
  dists <- dists[(nrow(centroids) + 1):nrow(dists), 1:nrow(centroids)]
  apply(dists, 1, which.min)}

clusters_B_from_A <- assign_clusters(partB, centroids_A)
full_clusters_B <- clusters[-train_idx]
agreement_table <- table(clusters_B_from_A, full_clusters_B)

Data Normalisation: Yes, normalization is mandatory

Euclidean distance is highly sensitive to scale. Variables such as sodium and fiber exhibit vastly different ranges. Without standardization, sodium and calories would dominate distance calculations and create distorted clusters. By applying scale to all numeric variables, every nutritional attribute contributes equally, resulting in the clear segments presented above.
